# 회의 생성 화면

## 화면 제목
회의 생성

## 입력 영역

### 회의 제목
- text input
- placeholder: 회의 제목을 입력하세요.

### 회의 목적
- text input 또는 text area
- placeholder: 회의 목적을 입력하세요.

### 회의 설명
- text area
- placeholder: 회의 배경, 참석자, 주요 안건 등을 입력하세요.

### 음성 파일 업로드
- file uploader
- 허용 형식: mp3, wav, m4a
- 안내 문구: 원본 음성 파일은 회의록 생성 후 저장하지 않습니다.

### 저장 위치
- radio
- 개인
- Teams

### 회의록 생성 버튼
- primary button
- 클릭 시 STT 및 회의록 생성 실행

## 처리 상태 영역
회의록 생성 버튼 클릭 후 단계별 상태를 표시한다.

- 입력값 검증
- 음성 텍스트 변환
- 회의록 생성
- 임시 파일 정리
- 미리보기 준비

## 회의록 편집 영역
AI가 생성한 회의록 markdown을 사용자가 직접 수정할 수 있는 영역이다.

구성:
- subheader: 회의록 편집
- text area
- height는 긴 회의록을 편집할 수 있도록 충분히 크게 설정
- text area의 기본값은 AI가 생성한 markdown 회의록

저장되는 내용:
- 사용자가 text area에서 최종 수정한 markdown 본문

## transcript 원문 영역
STT로 변환된 원문을 확인할 수 있는 영역이다.

구성:
- expander: 원문 보기
- transcript 텍스트 표시

정책:
- transcript는 기본적으로 저장하지 않는다.
- 화면에서 확인용으로만 제공한다.

## 저장 영역

### 회의록 저장 버튼
- 사용자가 회의록 편집을 완료한 뒤 클릭한다.
- 클릭 시 최종 markdown 내용을 파일로 저장한다.
- 자료관리 metadata에 회의록으로 등록한다.

## 저장 완료 메시지
저장이 완료되면 성공 메시지를 표시한다.

예:
- 회의록이 저장되었습니다.

## 자료관리 이동 후 확인
저장된 회의록은 자료관리 화면에서 확인한다.

- 개인 저장: 자료 관리 > 개인 > 회의록
- Teams 저장: 자료 관리 > Teams > 회의록
