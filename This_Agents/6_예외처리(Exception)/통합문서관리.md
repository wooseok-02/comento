# 설명 : 문서관리에서 발생할 수 있는 모든 것들을 미리 예상한다.

1. 지원하지 않는 파일 형식
- PDF, TXT, MD가 아닌 파일 업로드

2. 중복 문서
- 같은 내용 해시가 이미 존재

3. 텍스트 추출 실패
- PDF에서 텍스트를 추출할 수 없음
- 깨진 파일

4. 벡터화 실패
- 임베딩 생성 실패
- API 키 오류
- 임베딩 모델 호출 실패

5. 벡터 DB 저장 실패
- 저장 경로 문제
- 파일 권한 문제

6. 문서 조회 결과 없음
- 선택한 카테고리에 문서가 없음

7. 원본 파일 다운로드 실패
- metadata에는 있는데 원본 파일이 없음

8. 활성/비활성 변경 실패
- metadata 저장 실패

9. 벡터 DB 재생성 대상 없음
- 활성 상태이면서 처리 완료된 문서가 없음

10. 벡터 DB 재생성 중 일부 문서 실패
- 특정 문서만 텍스트 추출 실패
