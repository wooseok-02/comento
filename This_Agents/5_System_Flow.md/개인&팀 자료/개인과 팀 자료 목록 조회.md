1. 사용자가 사이드바에서 자료 관리 메뉴를 연다.
2. 사용자가 자료 범위를 선택한다.
   - personal
   - teams
3. 사용자가 자료 유형을 선택한다.
   - 전체
   - meeting_note
   - report
   - file
4. 시스템은 Supabase resources table을 조회한다.
5. 시스템은 owner_type과 resource_type이 일치하는 자료만 필터링한다.
6. 시스템은 created_at 기준 최신순으로 정렬한다.
7. 시스템은 자료 목록을 화면에 표시한다.
   - title
   - resource_type
   - source_type
   - created_at
   - 다운로드
8. 자료가 없으면 “등록된 자료가 없습니다.” 메시지를 표시한다.
