# 기본 패턴 : Chain + Parallel + 최소 Route

validate_input
→ retrieve_internal_documents ┐
                              → merge_context
→ search_web                  ┘
→ generate_report
→ format_sources
→ return_preview

# 설명:
validate_input
사용자가 입력한 제목, 주제, 목적을 검증한다.
입력값이 부족하면 그래프를 종료한다.

retrieve_internal_documents
Chroma Cloud vector store에서 관련 내부 문서를 검색한다.
결과는 internal_documents, internal_sources에 저장한다.

search_web
웹 검색을 실행한다.
결과는 web_results, web_sources에 저장한다.

merge_context
내부 문서와 웹 검색 결과를 보고서 생성용 context로 합친다.
결과는 merged_context, sources에 저장한다.

generate_report
LLM을 호출해 markdown 보고서를 생성한다.
결과는 report_content에 저장한다.

format_sources
내부 문서 출처와 웹 출처를 화면 표시 및 저장에 적합한 구조로 정리한다.

return_preview
UI에 보여줄 보고서 미리보기 결과를 반환한다.
