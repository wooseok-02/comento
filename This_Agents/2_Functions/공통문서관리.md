# 설명 : 요구사항을 만족하는 기능 목록들을 종합한다.

## 공통 자료 관리 기능 목록

1. 공통 문서 등록 기능
- 관리자가 PDF, TXT, MD 파일을 업로드한다.
- 카테고리를 선택한다.
- 파일명 기반으로 문서 제목이 생성된다.
- 파일 내용 해시로 중복 여부를 확인한다.

2. 문서 벡터 DB 자동 저장 기능
- 업로드된 문서를 텍스트로 추출한다.
- 문서를 청크 단위로 나눈다.
- 청크를 임베딩한다.
- 벡터 DB에 저장한다.

3. 문서 처리 상태 확인 기능
- 관리자가 문서 처리 상태를 확인한다.
- 상태는 처리중, 완료, 실패, 중복으로 표시한다.

4. 공통 문서 조회 기능
- 관리자가 등록된 문서 목록을 확인한다.
- 카테고리 기준으로 문서를 조회한다.

5. 공통 문서 비활성 처리 기능
- 관리자가 문서를 삭제 요청하면 비활성 상태로 변경한다.
- 비활성 문서는 AI 검색과 벡터 DB 재생성 대상에서 제외한다.

6. Vector DB 업데이트 기능
- 관리자가 수동으로 현재 활용 가능한 벡터들을 업데이트한다.
