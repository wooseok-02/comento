# 사용자 입력:
보고서 제목: 사용자가 직접 입력
보고서 주제: 사용자가 직접 입력
보고서 목적: 사용자가 직접 입력

# 저장 위치:
 개인 / Teams 선택

# 보고서 분량:
 기본값 고정

# 처리 방식:
내부 RAG 문서 검색
웹 검색
내부 문서와 웹 검색 결과를 함께 참고
AI가 보고서 생성
화면에 미리보기 표시
사용자가 저장 버튼을 누르면 저장

# 저장 방식:
보고서 파일은 Supabase Storage generated-files/reports/에 .md로 저장
자료관리에는 Supabase resources table에 등록
owner_type: personal 또는 teams
resource_type: report
source_type: generated_report
