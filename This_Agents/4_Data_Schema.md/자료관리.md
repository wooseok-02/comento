resource_id
- 개인/팀 자료 항목을 구분하는 고유 ID

owner_type
- personal 또는 teams
- 개인 자료인지 팀 자료인지 구분

resource_type
- meeting_note / report / file
- 회의록, 보고서, 일반 자료 구분

source_type
- common_document / generated_report / generated_meeting_note / uploaded_file
- 이 자료가 어디서 생겼는지 구분

title
- 화면에 보여줄 자료 제목

file_name
- 다운로드할 때 사용할 파일명

file_path
- 실제 파일 경로

source_document_id
- 공통 RAG 문서에서 가져온 경우 원본 document_id
- 생성된 보고서나 업로드 파일이면 None 가능

created_at
- 자료 저장 시점
