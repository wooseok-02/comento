# 대화방 관리와 대화방의 내용 저장

1. 대화방 관리

## 저장해야 하는 이유
- 사용자가 기존에 대화했던 기록을 확인한다.

## 필요 필드
chat_room_id
- 대화방을 고유하게 식별하기 위해 필요

title
- 사용자가 입력한 대화방 제목
- 사이드바 대화방 목록에 표시하기 위해 필요

created_at
- 대화방 생성 시점
- 나중에 생성순 관리나 이력 확인에 필요

updated_at
- 마지막 대화 시간
- 최근 대화순 정렬에 필요


2. 대화방의 내용 저장

## 저장해야 하는 이유
- 사용자가 기존에 대화했던 기록을 확인한다.

## 필요 필드
message_id
- 각 대화 메시지를 고유하게 식별하기 위해 필요

chat_room_id
- 이 메시지가 어떤 대화방에 속하는지 연결하기 위해 필요

role
- 메시지 작성자가 누구인지 구분하기 위해 필요
- user 또는 assistant

content
- 실제 질문 또는 답변 내용을 저장하기 위해 필요

sources
- AI 답변의 출처를 저장하기 위해 필요
- 사용자가 기존 대화방을 다시 열었을 때 답변 근거를 다시 확인하기 위해 필요

created_at
- 메시지가 생성된 시점
- 대화 내용을 순서대로 표시하기 위해 필요
