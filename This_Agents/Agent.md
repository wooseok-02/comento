# 최종 완성할 프로젝트 요구사항
ㅇ 생성형 AI 활용 검색 포털 서비스
- 자연어 분석 및 질의 의도 파악으로 최적의 답변이 가능한 AI 서비스
- 벡터 DB 및 인터넷 정보 활용하여 질의 관련 정보 제공 기능 구현
- 답변 작성 시 답변의 근거가 되는 원본 데이터 링크 표기
ㅇ 에이전트 기반의 지능형 업무 자동화 서비스
- 회의 음성 데이터 바탕으로 자동 회의록 작성 등 지능형 문서화 시스템
- 회의 내용을 자동으로 맞춤형 요약한 후 각 참석자에게 전송
- 메일, 회의록 분석을 통한 일정 관리 시스템
- 생성형 AI 기반의 OCR 시스템
- 벡터 DB 및 인터넷 정보를 활용하여 특정 주제에 대한 5페이지 이상 분량
의 보고서 작성 시스템
ㅇ AI 포털 관련 데이터 구축
- 규정, 내규, 법령, 판례, 보도자료 등 다양한 월드비전 내부 관련 정보 수집
및 체계화
- 수집된 데이터를 AI 학습 모델에 활용할 수 있도록 전처리 및 구조화
ㅇ AI 포털 사용자 기능
- 월드비전만의 AI 포털 제공(검색 서비스, 업무 자동화 서비스)
- 월드비전만의 내부 데이터 활용 RAG 시스템 구현
- 최근 대화 및 검색 기록 저장, 삭제 기능
- AI 모델 변경 기능
- 사용자 개인 자료 업로드 및 분석 기능
o 사용자 관리
- 사용자 추가/삭제
- 사용자 활성화/비활성화
- 사용자 그룹 추가/삭제
- 2차 인증 구현
o 로그 관리
- 생성형 AI으로 상담한 질문/답변의 사용자별 관리 기능
- 일자/모드/모델별 사용자수, 사용자별 질의횟수 등 관리
- 로그 다운로드 기능
o 비용 관리
- 모델별 사용자 비용 관리
- 비용 조회 결과 다운로드 기능
o 파일 관리
- 등록된 파일 관리 기능 (업로드／조회／삭제 등)
- 부서별 파일 관리
o 사용자 및 그룹별 데이터 접근 제어
o 서비스 관리자 기능 개발
- 답변 Temperature 조정 기능, 이용자 만족도 조사 기능 등
- 비정상적 이용자 및 매크로 사용자 차단 기능 등

# 사용할 기술
- LangChain, FAISS, Streamlit, RAG
